# 🍽️ Food Recipe

A modern and responsive recipe discovery web app built with **React.js** and powered by **TheMealDB API**. Search meals, explore categories, and view detailed recipes with ingredients, cooking instructions, and YouTube tutorials.

🌐 **Live Demo:** https://food-recipe-sigma-orpin.vercel.app/

---

## ✨ Features

- 🔍 Search recipes by name
- 🍱 Browse meal categories
- 🥕 Ingredients & measurements
- 👨‍🍳 Step-by-step cooking instructions
- 🎥 YouTube cooking tutorials
- 📱 Responsive design

---

## 🛠️ Tech Stack

- React.js
- Vite
- CSS3
- JavaScript
- TheMealDB API
- Vercel

---

## 🚀 Run Locally

```bash
git clone https://github.com/Spidy-Sudhir/Food-Recipe.git
cd Food-Recipe
npm install
npm run dev
```
